Finding the correct K-TAP version for your Linux kernel

You can use this application to find the appropriate K-TAP version for your Linux kernel.
  • Select the Guardium version
  • Select the required operating system
  • Optionally enter the kernel version - fully or partially

If you need to locate your Linux kernel version, use the command 'uname -r' in your Linux terminal or console.

You can download the latest KTAP modules from IBM Fix Central
