Matching the K-TAP Version to a Linux Kernel

You can use this application to find the appropriate K-TAP version for your Linux kernel.
  • Select the Guardium version
  • Select the required operating system
  • Optionally enter the kernel version - fully or partially

To locate the Linux kernel version, use the command 'uname -r' in your Linux terminal or console.
